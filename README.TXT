PROJECT TITLE: Improvement of CBC Encryption with Combination of Merkle - Hellman Knapsack Cryptosystem with Shifting Cipher Algorithm

PURPOSE OF PROJECT: To Improve the existing cryptosystem

VERSION or DATE: 5/5/2019

HOW TO START THIS PROJECT:

1.Dowload The JDE-BlueJ Version 4.2.1 from https://www.bluej.org/
2.Click on installer package and follow the installation steps
3.click on the BluJ icon on your desktop to run the project
4.after the BlueJ JDE is opened, Click on project>>open project>>select the cryptosystem/cryptosystem
5.Compile the Encryption_full-main class for encryption and Decryption_full-main class for Decryption
5.Right click on the Encryption_full main class, click on void main(String[] args)
6.The Terminal opens for Encryption
7.after which,Right click on the Decryption_full main class, click on void main(String[] args)
8.The Terminal opens for Decryption
